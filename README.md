# 📊 E-Commerce Sales Dashboard (Power BI & Excel)

## 📌 Project Overview

This project presents an **interactive dashboard** built using **Power BI and Excel** to analyze e-commerce sales performance.

It provides insights into **customer behavior, regional trends, and product performance**, helping identify opportunities for business growth.

---

## 📂 Dataset & Data Preparation

### 📁 Data Sources

* `orders.csv`
* `details.csv`

### ⚙️ Data Cleaning & Transformation

* Removed duplicates and inconsistent entries
* Handled missing values
* Standardized formats (dates, categories, values)
* Merged datasets using a common key

---

## 🛠️ Tools & Technologies

* **Microsoft Excel** – Initial data exploration
* **Power BI** – Data cleaning, transformation, modeling & visualization

---

## 📊 Dashboard Insights

### 🔹 Key Metrics

* **Total Sales:** 438K
* **Total Quantity Sold:** 5615
* **Total Profit:** 37K
* **Average Order Value (AOV):** 121K

---

### 🔹 Visualizations Included

* **Sales by State & Sub-Category** – Identifies top-performing regions and products
* **Quantity by Category** – Highlights most purchased categories
* **Profit by Month** – Tracks monthly profit trends
* **Quantity by Payment Mode** – Shows preferred payment methods
* **Top Customers by Sales** – Identifies high-value customers

---

### 🔹 Interactive Features

* Quarter filter
* State filter
* Dynamic visuals for deeper data exploration
